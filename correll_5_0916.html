Addison Correll's Journal 0916

Describe each of the following:

  • 1. ol, li - ol defines an ordered list and li defines the items in the list.
  • 2. ul - ul defines an unordered list.
  • 3. tr - tr defines a row in an HTML table.
  • 4. td - td defines a cell in an HTML table.
  • 5. a - a defines a hyperlink.
  • 6. aside - aside defines text that is aside from the main body of text.
  • 7. footer - footer defines text that is at the foot (the bottom) of the main body of text.
  • 8. margin-left, margin-right - margin defines the space around elements of an HTML, used in CSS.
  • 9. width - width defines the width of an element, used in CSS.
  • 10. height - height defines the height of an element, used in CSS.
  • 11. background-color - background-color defines the back ground color of an element, used in CSS.
  • 12. color - color defines the color of an element, used in CSS.
  • 13. a:hover - a:hover defines the color of an a tag when hovered over with the mouse, used in CSS.
  • 14. text-decoration - text-decoration defines if there are underlines, overlines, line-through or blinks of an element, used in CSS.
